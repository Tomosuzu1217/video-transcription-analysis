1. アプリの概要と全体像

1.1 このアプリでできること

本アプリは、動画CM（コマーシャル）の内容を自動で文字起こしし、AIを使って多角的に分析するためのWebアプリケーションです。マーケティング担当者やクリエイターが動画コンテンツの効果を科学的に把握し、改善に活かすことを目的としています。

主な用途: 動画の内容を自動テキスト化 → キーワード分析・心理分析・AI推奨 → コンバージョンとの相関把握 → 次回動画の改善提案を受ける

主要機能の一覧:

マーケティングレポートターゲット分析・競合優位性・改善提案・次回動画の方向性をAIが自動生成

2. 初回セットアップ

2.1 ログイン（パスワード認証）

アプリにアクセスすると、最初にパスワード入力画面が表示されます。これはチーム全員で共有する1つのパスワードです。

アプリURLにアクセス → パスワードを入力 → ダッシュボードが表示
補足:パスワードはブラウザに記憶されるため、次回以降は自動ログインされます。ログアウトは画面右上の「ログアウト」ボタンから行えます。

2.2 Gemini APIキーの登録

書き起こしやAI分析を利用するためには、Google Gemini APIキーが最低1つ必要です。

APIキーの取得手順

  1. ブラウザで Google AI Studio（https://aistudio.google.com）にアクセス
  2. Googleアカウントでログイン
  3. 左メニューの「Get API key」→「Create API key」をクリック
  4. 生成された文字列（例: AIzaSy...）をコピー

アプリへの登録手順

  1. ナビゲーションバーの「設定」をクリック
  2. 「Gemini APIキー管理」セクションの入力欄にコピーしたキーを貼り付け
  3. 「追加」ボタンをクリック
  4. 「全キーをテスト」ボタンで正常に登録できたか確認
ポイント:APIキーは複数登録可能です。一括書き起こし時にキーを並列で使用するため、複数登録するとより高速に処理できます（無料枠のレート制限を分散）。

2.3 AIモデルの選択

設定ページの「モデル選択」で、使用するGeminiモデルを選べます。

モデル名特徴おすすめ用途
gemini-2.5-flash高速・低コスト日常的な書き起こし・分析（デフォルト推奨）
gemini-2.5-pro高精度・やや低速重要な分析・レポート生成

2.4 タグの設定

設定ページの「タグ管理」で、動画を分類するためのタグを登録します。初期値として以下が設定されています：

YouTube / TikTok / Instagram / Facebook / LINE / X(Twitter)

自社で使用するSNSや分類に合わせて、タグの追加・削除が可能です。

3. ダッシュボード（トップページ）

ダッシュボードは、アプリのトップページとして機能する統合分析画面です。11個のタブに分かれており、それぞれ異なる角度からデータを分析できます。

タグ絞込:画面右上の「タグ絞込」ドロップダウンで、特定のSNS媒体（例: YouTube）に絞ったデータ表示が全タブに適用されます。

3.1 概要タブ

アプリ全体の状況を一目で把握できるダッシュボードです。

分析実行ボタン

画面右上の「分析を実行」ボタンをクリックすると、以下の4つの分析を自動で順番に実行します：

1. キーワード分析 → 2. 相関分析 → 3. AIレコメンデーション → 4. データ更新

実行中はステップインジケーターとプログレスバーが表示され、各ステップの進行状況と経過時間を確認できます。

サマリーカード（5枚）

カード表示内容
動画総数登録されている動画の総数
書き起こし完了書き起こしが完了した動画数 / 総数
処理中 / エラー現在処理中の動画数とエラーが発生した動画数
平均再生時間全動画の平均再生時間（MM:SS形式）
CV指標登録数コンバージョンデータが入力されている動画数

ストレージ使用量

Supabase Storageの使用量をバーで表示します（無料枠: 1GB）。動画ファイルのアップロード量に応じて増加します。

アラート通知

設定したアラートルールに該当するデータがある場合、赤い警告ボックスが表示されます。「アラート設定を確認 →」リンクからアラートタブに移動できます。

トップキーワードチャート

最新のキーワード分析結果から、上位15個のキーワードを横棒グラフで表示します。どのキーワードが動画内で多く使われているかが視覚的にわかります。

動画別コンバージョン比較チャート

各動画のコンバージョン指標を横棒グラフで並べて比較します。複数の指標がある場合は色分けされたバーで表示されます。

Top/Bottomパフォーマー

コンバージョン指標の上位3本（Topパフォーマー）と下位3本（要改善）を並べて表示します。

動画一覧テーブル

全動画をテーブル形式で一覧表示します。ファイル名、ステータス（書き起こし完了/処理中/エラー等）、再生時間、コンバージョン指標が確認できます。

AIレコメンデーション

最新のAI分析結果を表示します。概要サマリー、効果的なキーワード一覧（理由と出現動画付き）、改善提案（優先度バッジ付き）が含まれます。

3.2 動画比較タブ

2〜4本の動画を選択して、パフォーマンスを視覚的に比較します。

使い方

  1. コンバージョンデータのある動画がチップ（ボタン）で一覧表示されます
  2. 比較したい動画を2〜4本クリックして選択（青くなります）
  3. 自動でレーダーチャートと数値比較テーブルが生成されます

レーダーチャート

選択した動画の各指標を0〜100に正規化し、レーダー（蜘蛛の巣）チャートで重ねて表示します。動画ごとに色分けされ、どの指標が強い/弱いかが一目でわかります。

数値比較テーブル

全指標を行、選択した動画を列としたテーブルです。各行で最大値が青字太字でハイライトされます。

3.3 推移タブ

特定の指標の時系列変化を折れ線グラフで可視化します。

  1. 「指標」ドロップダウンから見たい指標（例: クリック率）を選択
  2. 動画の作成日順に折れ線グラフが描画されます
  3. 各データポイントにマウスを合わせると、動画名・日付・数値が表示されます

3.4 レポートタブ

Gemini AIが全データを統合分析し、マーケティング総合レポートを自動生成します。

レポート生成

  1. （任意）追加の分析指示をテキストエリアに入力（例:「20代女性をターゲットにした分析を重視」）
  2. 「レポート生成」ボタンをクリック
  3. AIが分析中はスピナーが表示されます（通常30秒〜2分）

レポートの内容

セクション内容
エグゼクティブサマリー分析結果の要約（1段落）
ターゲット分析想定されるターゲット層とそれぞれへのキーメッセージ
コンテンツ評価マトリクス各動画の10点満点スコア・強み・弱み
競合優位性自社コンテンツの競合に対する強みとその根拠
改善優先度改善すべき領域（高/中/低の優先度）、推奨アクション、期待効果
次回動画の方向性推奨テーマ・構成・感情曲線・キーメッセージ

エクスポート

  • HTMLエクスポート: ブラウザで開けるHTML形式のレポートファイルをダウンロード
  • PPTXエクスポート: PowerPoint形式のプレゼンテーションファイルをダウンロード

AI台本提案

「AI台本提案」ボタンをクリックすると、過去の分析結果に基づいて次回動画の台本構成案をAIが提案します。台本構成、推奨構成、タイミングガイド、感情曲線、キーメッセージ、参考動画が含まれます。

3.5 ROIタブ

動画広告の投資対効果を計算・可視化します。

使い方

  1. 「収益指標」ドロップダウンから売上に相当する指標を選択（例: 売上額）
  2. 「コスト指標」ドロップダウンから費用に相当する指標を選択（例: 広告費）
  3. 自動で以下が計算・表示されます：
指標計算式意味
ROI(売上 - コスト) / コスト × 100投資に対する利益率。100%なら投資額の2倍の売上
ROAS売上 / コスト広告費1円あたりの売上。2.0なら2倍の売上
CPAコスト / コンバージョン数1コンバージョンあたりの獲得コスト

3.6 ファネルタブ

マーケティングファネル（顧客が購入に至るまでの段階的な流れ）を自由に構築・可視化します。

使い方

  1. 「指標を追加」ドロップダウンから段階ごとの指標を選択（例: インプレッション→クリック→購入）
  2. （任意）動画を絞り込むか、全動画の合計で表示するか選択
  3. ステージは上下矢印ボタンで並び替え可能
  4. 横棒グラフで各段階の値と通過率（コンバージョン率）が表示されます

3.7 競合比較タブ

競合他社のベンチマーク数値を登録し、自社動画と比較します。

  1. 「競合を追加」フォームで競合名と各指標の数値を入力
  2. 登録した競合データと自社動画の指標が同じグラフ上に表示されます
  3. 競合データは編集・削除が可能です

3.8 アラートタブ

指標が指定した閾値を超えた（または下回った）場合に自動で警告を出すアラートルールを設定します。

アラート作成

設定項目説明
指標監視する指標名（ドロップダウンから選択）
条件「以上（above）」または「以下（below）」
閾値基準となる数値
動画（任意）特定の動画のみを対象にする場合に選択
有効/無効アラートの有効・無効を切り替え

アラートが発動すると、概要タブの上部に赤い通知が表示されます。

3.9 媒体分析タブ

SNSプラットフォーム（YouTube / TikTok / Instagram 等）ごとに、AI がコンテンツの最適化提案を行います。

分析内容

分析項目内容
プラットフォーム特性最適な動画長、効果的なフック手法、ストーリーテリングパターン
トーン&スタイル各媒体に適した雰囲気・語彙・テンポのガイドライン
CTA戦略プラットフォーム別の行動喚起（Call to Action）最適化提案
ベスト/ワースト動画該当媒体で高い/低いパフォーマンスの動画と改善ギャップ
クロスプラットフォーム提案他媒体用にコンテンツを再利用する際のアダプテーション案

3.10 ランキングインサイトタブ

動画のランキング（順位）とプラットフォームを掛け合わせたマトリクス分析です。

分析内容

  • プラットフォーム-ランキングマトリクス: 各動画がどの媒体でどの程度効果的かを一覧化
  • ペルソナプロファイル: 各プラットフォームのターゲット層のデモグラフィックス・行動パターン
  • ヒットファクター分析: 成功動画に共通する要因を「重要度（高/中/低）」で分類
  • コンテンツ戦略: 媒体別の理想的な動画長・フック戦略・ペルソナメッセージング・CTA手法・台本概要

3.11 分析履歴タブ

過去に実行した全ての分析結果を時系列で閲覧できます。

表示項目内容
日時分析が実行された日時
分析タイプキーワード分析、相関分析、AI推奨、マーケティングレポート等
モデル使用されたGeminiモデル名
スコープ分析の範囲（全体/特定動画など）

各行をクリックすると、分析結果のJSON詳細が展開表示されます。ページネーション機能付き（20件/ページ）。

4. 動画管理ページ

動画ファイルのアップロード、一覧管理、書き起こし処理を行うページです。

4.1 動画のアップロード

対応ファイル形式

動画ファイル

MP4, WebM, MOV, AVI, MKV, FLV, WMV, M4V

音声ファイル

MP3, WAV, AAC, OGG, FLAC, WMA, M4A, Opus

アップロード手順

  1. ページ上部のドラッグ&ドロップエリアにファイルをドラッグ、またはクリックしてファイルを選択
  2. 複数ファイルを同時に選択可能
  3. アップロードの進行状況がプログレスバーで表示されます
  4. 完了すると動画一覧に自動で追加されます
容量制限:Supabase無料枠の制約により、1ファイルあたり最大50MB、ストレージ合計最大1GBです。動画は圧縮してからアップロードすることをお勧めします。

4.2 書き起こし（文字起こし）

アップロードした動画の音声をGemini AIがテキストに変換します。

個別の書き起こし

  1. 動画一覧からカードをクリックして詳細ページを開く
  2. 「書き起こし開始」ボタンをクリック
  3. 処理中はステータスが「書き起こし中」に変わり、進行状況が表示されます
  4. 完了するとテキストとタイムスタンプ付きセグメントが表示されます

4.3 一括書き起こし

まだ書き起こしされていない動画をまとめて処理できます。

  1. 「アップロード済」ステータスの動画がある場合、画面上部に「N本の動画が未書き起こし」という通知が表示されます
  2. 「一括書き起こし」ボタンをクリック
  3. バッチ進行パネルが表示され、各動画の処理状況がリアルタイムで更新されます：
ステータス意味
待機中（queued）処理待ち
ダウンロード中ストレージから動画を取得中
準備中APIへの送信準備中
書き起こし中Gemini AIが音声認識を実行中
保存中結果をデータベースに保存中
完了書き起こし成功
エラー処理に失敗（APIキー制限等）

複数のAPIキーを登録している場合、並列処理されるため高速です。APIキーの状態（利用可能/使用中/レート制限）も表示されます。

4.4 動画の検索・フィルター

  • ファイル名検索: テキスト入力で動画をフィルター（部分一致、大文字小文字区別なし）
  • タグフィルター: タグチップ（YouTube, TikTok等）をクリックして特定タグの動画のみ表示

4.5 タグ管理

各動画カードにタグ選択UIがあり、ドロップダウンから複数のタグを付与できます。タグは設定ページで事前に定義したものが表示されます。

4.6 CSVエクスポート

「CSV出力」ボタンで、現在表示中の動画リストをCSVファイルとしてダウンロードできます。ファイル名、ステータス、再生時間、タグ等の情報が含まれます。

5. 動画詳細ページ

動画一覧からカードをクリックすると開く、個別動画の詳細画面です。

5.1 動画再生

  • HTML5ビデオプレーヤーで動画を再生できます
  • 再生速度の変更: 0.5倍〜2倍速に調整可能
  • タイムライン連動: 書き起こしのセグメント（時間区間）をクリックすると、その時点から再生が開始されます

5.2 書き起こし結果の閲覧・編集

全文表示

書き起こし結果がテキストエリアに表示されます。直接編集が可能で、「保存」ボタンで変更を反映できます。

セグメント表示

書き起こしテキストは時間区間（セグメント）ごとに分割されて表示されます。

  • 各セグメントには開始時間〜終了時間とテキストが表示されます
  • セグメントをクリックすると、動画がその時点にジャンプして再生されます
  • 各セグメントのテキストを個別に編集・保存できます

5.3 書き起こしのエクスポート

書き起こし結果を以下の形式でダウンロードできます：

形式用途ファイル拡張子
テキストプレーンテキストとして利用.txt
SRT字幕ファイル（動画編集ソフトで読込可能）.srt
VTTWeb用字幕ファイル.vtt
JSONプログラムでの利用・データ保存.json

5.4 ランキング・メタデータ編集

  • ファイル名の変更: ファイル名横の編集ボタンから変更可能
  • ランキング設定: 数値（1位、2位...）を入力して動画に順位を付けられます。補足メモも入力可能
  • タグの変更: マルチセレクトで複数タグの付与・解除

5.5 コンバージョン指標の管理

動画ごとにパフォーマンス指標を記録します。

指標の追加

  1. 「指標名」に名前を入力（例: 再生回数、クリック率、CV率）
  2. 「値」に数値を入力
  3. （任意）記録日、メモを入力
  4. 「追加」ボタンで保存

登録した指標はダッシュボードの比較・推移・ROI等の分析に自動で反映されます。各指標は編集・削除も可能です。コンバージョンデータのCSVエクスポートもできます。

5.6 アーカイブ

「アーカイブ」ボタンで動画をアーカイブ状態にできます。アーカイブすると元の動画ファイルがストレージから削除されますが、書き起こしテキストとメタデータは保持されます。ストレージ容量を節約したい場合に有効です。

6. 分析ページ

5つのタブで構成される、動画コンテンツの詳細分析を行うページです。

6.1 キーワード分析タブ

全動画の書き起こしテキストから頻出キーワードを抽出し、ランキング化します。

媒体フィルター（新機能）

「媒体」ドロップダウンで、分析対象をSNS別に絞り込めます。

  • 全体: 全動画を対象に分析
  • YouTube / TikTok / Instagram 等: 該当タグが付いた動画のみを対象に分析

分析結果

  • トップ20棒グラフ: 上位20キーワードを横棒グラフで可視化
  • トップ50テーブル: 順位、キーワード、出現回数、該当動画名を一覧表示
  • CSVエクスポート: 分析結果をCSVファイルとしてダウンロード可能

6.2 相関分析タブ

キーワードの出現とコンバージョン率の間に統計的な相関があるかを分析します。

分析結果

  • 散布図チャート: X軸＝効果スコア、Y軸＝平均CV率、バブルサイズ＝動画数
  • 相関テーブル: キーワードごとに以下を表示：
    • キーワードあり時の平均CV率
    • キーワードなし時の平均CV率
    • 効果スコア（差分）
    • 該当動画数
  • CSVエクスポート: 相関データをCSVでダウンロード
読み方:効果スコアが正の大きな値のキーワードは、コンバージョン率を高める可能性があるキーワードです。逆に負の値は、CV率の低い動画に多く含まれるキーワードです。

6.3 AI総合分析タブ

4つの分析機能を1つのタブに統合した、最も強力な分析機能です。

テキスト検索

上部の検索バーで、全動画の書き起こしテキストから全文検索ができます。検索結果には動画名とマッチしたテキスト部分が表示されます。

AI総合分析の実行

「AI総合分析を実行」ボタンをクリックすると、以下の3つの分析が同時並行で実行されます：

分析名内容
AI推奨 書き起こしとCV データを総合分析し、効果的なキーワード・改善提案・ファネル改善案を生成
ランキング比較 ランキング上位と下位の動画を比較し、心理分析・ストーリーテリング・言語テクニック・差異を詳細分析
コンテンツ心理分析 心理学的分析手法に基づき、感情ボラティリティ・ストーリーテリング手法・CV導線の分析を実行

カスタムプロンプト

「追加指示」テキストエリアに自由な分析指示を入力できます。例：「美容系商材のターゲットに絞った分析をして」「冒頭30秒のフック手法を重点的に」

結果の表示

各分析結果は折りたたみ可能なセクションで表示されます。セクション右上に「最終実行日時」が表示され、前回の分析結果もいつでも確認できます。

6.4 書き起こし一覧タブ

全動画の書き起こしテキストを一覧表示します。

  • 各動画ごとに折りたたみ式のセクションで表示
  • 動画名、セグメント数、再生時間、言語が確認できます
  • 展開すると全文テキストとセグメント一覧（タイムスタンプ付き）を閲覧可能
  • 「詳細を見る」リンクから動画詳細ページに移動できます

6.5 分析履歴タブ

ダッシュボードの分析履歴タブと同様に、過去の全分析結果を時系列で閲覧・展開できます。20件ごとのページネーション付き。

7. 設定ページ

7.1 APIキー管理

  • キーの追加: テキスト入力欄にAPIキーを貼り付け、「追加」をクリック
  • キーの一覧: 登録済みキーはマスク表示（セキュリティのため先頭数文字のみ表示）
  • キーの削除: 各キーの「削除」ボタンで除去（確認ダイアログあり）
  • 全キーテスト: 全登録キーを順にテストし、有効/無効を判定。エラーの場合はエラーメッセージを表示

7.2 モデル選択

ドロップダウンメニューから使用するGemini AIモデルを選択します。カスタムモデル名の入力も可能です。選択は即座に保存されます。

7.3 ストレージ使用量

Supabase Storageの使用量をバーグラフで表示します。使用済み容量（MB）と上限（1GB）が数値で確認できます。

7.4 タグ管理

  • タグの追加: テキスト入力欄にタグ名を入力して「追加」（重複は不可）
  • タグの削除: 各タグチップの「×」ボタンで削除
  • 追加・削除は即座にデータベースに保存されます
  • ここで管理するタグが、動画管理ページやキーワード分析の媒体フィルターに反映されます

9. よくある質問（FAQ）

Q. APIキーは有料ですか？

A. Google Gemini APIには無料枠があります。個人利用や小規模チームであれば、通常は無料枠内で収まります。大量の動画を処理する場合は有料プランが必要になる可能性があります。

Q. 書き起こしが「エラー」になりました。どうすれば？

A. 主な原因はAPIキーのレート制限です。以下を試してください：

  1. 数分待ってからリトライ（動画詳細ページの「リトライ」ボタン）
  2. 設定ページで「全キーをテスト」してキーが有効か確認
  3. 複数のAPIキーを登録してレート制限を分散

Q. 動画をアップロードできません

A. ファイルサイズが50MB以下か確認してください。また、ストレージ上限（1GB）に達していないか、設定ページのストレージ使用量を確認してください。

Q. 分析結果が空です

A. 分析には書き起こし済みの動画が必要です。まず動画の書き起こしを完了させてから分析を実行してください。

Q. 相関分析で結果が出ません

A. 相関分析には、書き起こし済みの動画に加えてコンバージョン指標が登録されている必要があります。動画詳細ページからCV指標を入力してください。

Q. タグはどう使い分ければよいですか？

A. 主な使い方は投稿先SNSでの分類です。例えばYouTube用の動画には「YouTube」タグ、TikTok用には「TikTok」タグを付けることで、媒体別の分析が可能になります。

Q. ダークモードはありますか？

A. はい。ナビゲーションバー右側の太陽/月アイコンをクリックすると、ライトモードとダークモードを切り替えられます。設定はブラウザに記憶されます。

Q. データは安全ですか？

A. データはSupabase（クラウドデータベース）に暗号化通信（HTTPS）で保存されます。アプリはパスワード認証で保護されています。ただし、高度なセキュリティが必要な場合は追加対策を検討してください。
